# 合同管理系统 (轻量版)

基于 Python tkinter 开发的轻量级合同管理系统，打包后体积约 **15-20MB**。

## 功能特性

### 1. 合同管理
- 录入合同基本信息（合同编号、项目代码、对方单位、区域等）
- 记录合同金额、起始日期、终止日期
- 标注合同类型（维保/维修/技术服务等）

### 2. 开票登记
- 选择合同后录入开票日期和金额
- 支持多次开票，自动累计

### 3. 回款登记
- 选择合同后录入回款日期和金额
- 支持多次回款，自动累计

### 4. 统计分析
- **年度统计**：展示每年的合同数量、合同总额、开票总额、回款总额、应收账款
- **区域统计**：按区域展示合同分布、开票回款情况
- **应收账款**：自动计算开票金额与回款金额的差额

### 5. 数据导入
- 支持从 CSV 文件导入合同数据

## 技术特点

- **轻量级**：仅使用 Python 内置库（tkinter + sqlite3），无额外依赖
- **体积小**：打包后约 15-20MB（相比 PyQt5 版本的 80-120MB）
- **跨平台**：支持 Windows、macOS、Linux

## 快速开始

### 方式一：直接运行 Python 脚本

```bash
# 无需安装依赖，直接运行
python main.py
```

### 方式二：打包为 exe 文件

#### 在 Windows 上打包

1. 双击运行 `build.bat`
2. 等待打包完成
3. 在 `dist` 文件夹中找到 `合同管理系统.exe`

#### 使用 GitHub Actions 自动打包

1. 将代码上传到 GitHub 仓库
2. 进入 Actions 页面
3. 等待自动打包完成
4. 下载生成的 exe 文件

## CSV 导入格式

CSV 文件应包含以下列：

```csv
合同编号,项目代码,对方单位名称,区域,合同金额,合同起始日期,合同终止日期,合同内容
FMD26001,XM2026001,某某公司,华北区,100000,2026-01-01,2026-12-31,维保费
```

## 系统要求

- Python 3.8+（如直接运行）
- Windows 7+ / macOS 10.12+ / Linux（如运行打包后的 exe）

- 数据库文件：`contracts.db`（SQLite）
